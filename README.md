# Musk 5-Step Improvement Skill

This skill applies Elon Musk's 5-step algorithm to improve any system,
process, product, or codebase.

It:
- Questions and cleans up requirements
- Deletes unnecessary parts or steps
- Simplifies and optimizes what remains
- Accelerates cycle time and feedback loops
- Only then suggests automation for stable, repetitive parts

## Installation

With the Skills CLI:

```bash
npx skills add mavko/musk-5-step-improvement
```

## Usage

Call the `musk-5-step-improvement` skill from your AI agent when you want
to redesign or optimize a process, codebase, or product.

The skill will:
1. Ask you to describe the current system and goals
2. Walk through the 5 steps in order
3. Output:
   - Requirements, Deletions, Simplifications, Cycle Time, Automation
   - A "New Version v1.0" summary
   - An implementation checklist (Now / Next / Later)
